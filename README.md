# homebrew-safehouse

Homebrew tap for [Agent Safehouse](https://github.com/eugene1g/agent-safehouse).

## Install

Stable releases:

```bash
brew install eugene1g/safehouse/agent-safehouse
```

Head builds from `main`:

```bash
brew install --HEAD eugene1g/safehouse/agent-safehouse
```

## Maintenance

Stable formula updates are published from the main repository with:

```bash
./scripts/publish-homebrew-tap.sh vX.Y.Z --push
```
